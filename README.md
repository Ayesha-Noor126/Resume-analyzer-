# 🧠 AI CV Evaluator

An intelligent full-stack web application that semantically analyzes a CV against a Job Description — scoring the match section-wise, identifying skill gaps, and generating a personalized AI evaluation report.



---

## 📌 Project Overview

Upload a CV (PDF or DOCX) and paste a Job Description. The system returns:

- **ATS Match Score** — weighted semantic similarity across CV sections
- **Section Breakdown** — Skills, Experience, Projects, Education scored individually
- **Missing Skills** — JD keywords absent from the CV
- **AI Evaluation** — 3–4 sentence report generated by Llama 3.3 70B
- **Recommendations** — 5 actionable improvement suggestions
- **RAGAS Scores** — pipeline quality evaluation (Faithfulness, Answer Relevancy, Context Precision)
- **PDF Export** — full report downloadable as a styled PDF

## 🔬 Custom RAGAS Evaluation

The standard RAGAS library was incompatible with Groq's `n=1` completion enforcement — RAGAS internally requests `n>1` for statistical reliability, causing a 400 error and zeroing out all metrics. All three metrics were rebuilt from scratch:

### Faithfulness
Measures whether the AI evaluation paragraph is grounded in the actual CV — not hallucinating skills or experience. Uses a single structured Groq LLM call to extract every factual claim from the report and verify each against the CV text.

```
Score = supported claims / total claims
```

### Answer Relevancy
Measures whether the generated report actually addresses the job description requirements. Computed as pure embedding cosine similarity between the JD and the evaluation paragraph — no LLM call needed.

```
Score = cosine_similarity(embed(JD), embed(evaluation_paragraph))
```

### Context Precision
Measures how precisely the retrieved CV chunks match the JD. Splits the CV into 150-word chunks, embeds all chunks and the JD, takes the top 40% highest-similarity chunks as "retrieved", then scores what fraction exceed a relevance threshold of 0.4.

```
Score = relevant chunks in top-k / k
```

### Results on a Strong Match (Full Stack Dev CV vs matching JD)

| Metric | Score |
|---|---|
| Faithfulness | 89% |
| Answer Relevancy | 86% |
| Context Precision | 100% |
| **RAGAS Score** | **93%** |

## 🚀 Local Setup

### Prerequisites
- Python 3.13
- Node.js 18+
- Free Groq API key — [console.groq.com](https://console.groq.com)

### Backend
```bash
cd backend
pip install -r requirements.txt
```



```bash
python -m uvicorn main:app --reload
# http://localhost:8000
# First run downloads BAAI/bge-small-en-v1.5 (~130MB, cached after)
```

### Frontend
```bash
cd frontend
npm install
npm run dev
# http://localhost:5173
```
